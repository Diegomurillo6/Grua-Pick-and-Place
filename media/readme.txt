In this link you may see a small demonstration of the crane's working principle:
https://youtu.be/kWXMplV7dBI

In the video, which is like 8x, due to speed limitations with educational-level actuators, the system initially detects that all stacks are full, so no pick operation is performed until space becomes available. Once space is cleared in a stack, the crane moves to the pick position and reads the QR code of the box to determine its destination. If the corresponding stack is still full, the system waits until space is available. When a valid condition is met, the crane performs the pick-and-place operation and returns to a standby state while waiting for new items.
